Key architectural differences from NVIDIA:
- **24 Tensor Processing Cores (TPCs)** vs NVIDIA's streaming multiprocessors
- **Large on-chip SRAM (48MB)** vs NVIDIA's ~40MB L2 cache
- **Graph-based compilation** vs CUDA's dynamic kernel launch model

## TPC Memory Access Optimization

Each TPC has 256KB local SRAM, vector and scalar execution units, and hardware prefetch capabilities. The key to performance is loading data into local SRAM and reusing it:

```cpp
// Optimized TPC kernel: cache vector chunk in local SRAM for reuse
void tpc_optimized_matvec(
    __global__ float* __restrict__ matrix,  // [M, K]
    __global__ float* __restrict__ vector,  // [K]
    __global__ float* __restrict__ output,  // [M]
    int M, int K
) {
    __local__ float vec_cache[256];  // 256 floats fit in local SRAM
    int row_start = get_tpc_id() * ROWS_PER_TPC;
    int row_end = min(row_start + ROWS_PER_TPC, M);

    for (int k_base = 0; k_base < K; k_base += 256) {
        int k_end = min(k_base + 256, K);
        // Collaborative load: HW broadcast optimization kicks in
        async_load_256b_aligned(&vec_cache[0], &vector[k_base], k_end - k_base);
        barrier();

        for (int row = row_start; row < row_end; row++) {
            float sum = 0.0f;
            #pragma unroll 8
            for (int k = 0; k < k_end - k_base; k++) {
                sum += matrix[row * K + k_base + k] * vec_cache[k];
            }
            output[row] += sum;
        }
    }
}
```

## Graph Compilation Optimization

Unlike CUDA's dynamic kernel launches, Gaudi2 compiles entire computation graphs ahead of time. This enables cross-layer memory optimization:

```python
import habana_frameworks.torch as ht

def optimize_inference_graph(model, sample_input):
    """Configure graph compilation for optimal memory usage."""
    ht.hpu.enable_inference_mode()

    # Level 3: Cross-layer optimization (longer compile, better memory reuse)
    ht.core.hpu_set_env("PT_HPU_LAZY_MEMORY_OPTIMIZATION_LEVEL", "3")
    ht.core.hpu_set_env("PT_HPU_ENABLE_SRAM_OFFLOAD", "1")
    ht.core.hpu_set_env("PT_HPU_MAX_SRAM_PER_GRAPH_MB", "40")

    with torch.no_grad():
        model(sample_input)      # Triggers compilation
        ht.hpu.synchronize()
        model(sample_input)      # Uses cached graph
        ht.hpu.synchronize()

    return model
```
