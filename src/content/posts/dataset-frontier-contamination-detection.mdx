If a language model's training data contains benchmark test examples, its performance on that benchmark is meaningless. The model is not demonstrating generalization -- it is recalling memorized answers. This is data contamination, and it is the most pervasive integrity problem in LLM evaluation.

The challenge: training datasets contain trillions of tokens scraped from the internet. Benchmarks are published on the internet. The overlap is not zero. Detecting contamination requires comparing the training set against all evaluation benchmarks and quantifying the degree of leakage. Prevention requires designing evaluation protocols that are robust against contamination.

This post covers detection methods (n-gram overlap, perplexity-based, membership inference), prevention strategies (private eval sets, dynamic benchmarks, canary strings), and a complete contamination scanner implementation.

## 1. Why Contamination Matters

### The Scale of the Problem

A study by Sainz et al. (2023) found that GPT-4 had been contaminated on portions of at least 7 of 12 evaluated benchmarks. Dodge et al. (2021) found that C4, one of the most commonly used pretraining corpora, contains exact copies of examples from GLUE, SuperGLUE, and SQuAD test sets.

### Types of Contamination

**Direct contamination**: The exact test example (question + answer) appears in training data. The model memorizes the answer.

**Indirect contamination**: A paraphrase, discussion, or solution to the test example appears in training data. The model has seen the problem in a different form.

**Partial contamination**: The question appears but the answer does not (or vice versa). The model has unfair priming but must still reason.

```python
class ContaminationType:
    DIRECT = "direct"          # Exact match of test example in training data
    INDIRECT = "indirect"      # Paraphrase or discussion of test example
    PARTIAL = "partial"        # Question or answer appears, not both
    NONE = "none"              # No contamination detected

def classify_contamination(test_example, training_match, overlap_threshold=0.8):
    """Classify the type of contamination."""
    q_overlap = compute_ngram_overlap(
        test_example["question"], training_match["text"], n=8
    )
    a_overlap = compute_ngram_overlap(
        test_example["answer"], training_match["text"], n=8
    )

    if q_overlap > overlap_threshold and a_overlap > overlap_threshold:
        return ContaminationType.DIRECT
    elif q_overlap > 0.5 and a_overlap > 0.5:
        return ContaminationType.INDIRECT
    elif q_overlap > overlap_threshold or a_overlap > overlap_threshold:
        return ContaminationType.PARTIAL
    else:
        return ContaminationType.NONE
```

## 2. Detection Method 1: N-gram Overlap

### The Algorithm

The simplest contamination check: compare n-grams between benchmark examples and training data. If a long n-gram from a test example appears verbatim in training data, contamination is likely.

```python
from collections import defaultdict
import hashlib

class NGramContaminationDetector:
    """Detect contamination using n-gram overlap."""

    def __init__(self, n=13, min_overlap_ratio=0.8):
        self.n = n
        self.min_overlap_ratio = min_overlap_ratio
        self.training_ngrams = set()  # Set of training n-gram hashes

    def index_training_data(self, training_texts):
        """Build n-gram index from training data."""
        for text in training_texts:
            tokens = self._tokenize(text)
            for i in range(len(tokens) - self.n + 1):
                ngram = tuple(tokens[i:i + self.n])
                ngram_hash = hashlib.md5(
                    " ".join(ngram).encode()
                ).hexdigest()
                self.training_ngrams.add(ngram_hash)

    def check_contamination(self, test_example):
        """Check if a test example is contaminated."""
        text = test_example.get("question", "") + " " + test_example.get("answer", "")
        tokens = self._tokenize(text)

        if len(tokens) < self.n:
            return {"contaminated": False, "overlap_ratio": 0.0}

        total_ngrams = len(tokens) - self.n + 1
        matching_ngrams = 0

        for i in range(total_ngrams):
            ngram = tuple(tokens[i:i + self.n])
            ngram_hash = hashlib.md5(
                " ".join(ngram).encode()
            ).hexdigest()

            if ngram_hash in self.training_ngrams:
                matching_ngrams += 1

        overlap_ratio = matching_ngrams / total_ngrams

        return {
            "contaminated": overlap_ratio >= self.min_overlap_ratio,
            "overlap_ratio": overlap_ratio,
            "matching_ngrams": matching_ngrams,
            "total_ngrams": total_ngrams,
        }

    def scan_benchmark(self, benchmark_examples):
        """Scan an entire benchmark for contamination."""
        results = {
            "total": len(benchmark_examples),
            "contaminated": 0,
            "partial": 0,
            "clean": 0,
            "details": [],
        }

        for ex in benchmark_examples:
            result = self.check_contamination(ex)
            result["example_id"] = ex.get("id", "unknown")

            if result["contaminated"]:
                results["contaminated"] += 1
            elif result["overlap_ratio"] > 0.3:
                results["partial"] += 1
            else:
                results["clean"] += 1

            results["details"].append(result)

        results["contamination_rate"] = results["contaminated"] / results["total"]
        return results

    @staticmethod
    def _tokenize(text):
        """Simple whitespace tokenization with lowercasing."""
        return text.lower().split()
```

### Choosing N

The n-gram length $n$ controls the sensitivity-specificity tradeoff:

- Small $n$ (5-8): High recall but many false positives. Common phrases match coincidentally.
- Medium $n$ (10-15): Good balance. 13-grams are the most common choice.
- Large $n$ (20+): Low false positives but misses paraphrased contamination.

## 3. Detection Method 2: Perplexity-Based

### The Intuition

If a model has seen a test example during training, it will assign that example lower perplexity (higher probability) than similar examples it has not seen. We compare the model's perplexity on test examples against a reference distribution.

```python
import torch
import numpy as np

class PerplexityContaminationDetector:
    """Detect contamination using model perplexity analysis."""

    def __init__(self, model, tokenizer, z_threshold=2.5):
        self.model = model
        self.tokenizer = tokenizer
        self.z_threshold = z_threshold

    def compute_perplexity(self, text):
        """Compute the model's perplexity on a text."""
        tokens = self.tokenizer.encode(text, return_tensors="pt").to("cuda")

        with torch.no_grad():
            outputs = self.model(tokens, labels=tokens)
            loss = outputs.loss.item()

        return np.exp(loss)

    def detect_contamination(self, test_examples, reference_examples):
        """
        Compare perplexity on test examples vs. reference examples.

        Test examples with anomalously low perplexity (relative to reference)
        are likely contaminated.
        """
        # Compute reference perplexity distribution
        ref_ppls = [
            self.compute_perplexity(ex["text"])
            for ex in reference_examples
        ]
        ref_mean = np.mean(ref_ppls)
        ref_std = np.std(ref_ppls)

        results = []
        for ex in test_examples:
            ppl = self.compute_perplexity(ex["text"])

            # Z-score: how many standard deviations below the reference mean
            z_score = (ref_mean - ppl) / max(ref_std, 1e-6)

            contaminated = z_score > self.z_threshold

            results.append({
                "example_id": ex.get("id", "unknown"),
                "perplexity": ppl,
                "z_score": z_score,
                "contaminated": contaminated,
            })

        return results

    def detect_with_canonicalization(self, test_examples):
        """
        Compare perplexity of original vs. canonicalized (rephrased) versions.
        Contaminated examples will have much lower perplexity in original form.
        """
        results = []
        for ex in test_examples:
            original_ppl = self.compute_perplexity(ex["text"])
            canonical_ppl = self.compute_perplexity(ex["canonical_text"])

            # Ratio > 1 means original has lower perplexity (suspicious)
            ratio = canonical_ppl / max(original_ppl, 1e-6)

            results.append({
                "example_id": ex.get("id"),
                "original_ppl": original_ppl,
                "canonical_ppl": canonical_ppl,
                "ppl_ratio": ratio,
                "contaminated": ratio > 1.5,
            })

        return results
```

### Min-K% Prob Method

Shi et al. (2023) propose a method that looks at the minimum-probability tokens -- the "hardest" tokens for the model to predict. If the model has memorized an example, even the hardest tokens will have relatively high probability:

```python
class MinKProbDetector:
    """
    Min-K% Prob contamination detection.
    Looks at the k% lowest-probability tokens in a sequence.
    """

    def __init__(self, model, tokenizer, k_percent=20):
        self.model = model
        self.tokenizer = tokenizer
        self.k_percent = k_percent

    def compute_min_k_prob(self, text):
        """Compute the average log-probability of the bottom k% tokens."""
        tokens = self.tokenizer.encode(text, return_tensors="pt").to("cuda")

        with torch.no_grad():
            outputs = self.model(tokens)
            logits = outputs.logits

        # Compute per-token log probabilities
        log_probs = torch.log_softmax(logits[0, :-1], dim=-1)
        target_tokens = tokens[0, 1:]
        token_log_probs = log_probs.gather(1, target_tokens.unsqueeze(1)).squeeze()

        # Get the bottom k% (lowest probability tokens)
        k = max(1, int(len(token_log_probs) * self.k_percent / 100))
        sorted_probs, _ = torch.sort(token_log_probs)
        min_k_avg = sorted_probs[:k].mean().item()

        return min_k_avg

    def detect(self, test_examples, reference_examples):
        """Detect contamination using Min-K% Prob."""
        ref_scores = [self.compute_min_k_prob(ex["text"]) for ex in reference_examples]
        ref_mean = np.mean(ref_scores)
        ref_std = np.std(ref_scores)

        results = []
        for ex in test_examples:
            score = self.compute_min_k_prob(ex["text"])
            z_score = (score - ref_mean) / max(ref_std, 1e-6)

            results.append({
                "example_id": ex.get("id"),
                "min_k_score": score,
                "z_score": z_score,
                # Higher score (less negative) means memorized
                "contaminated": z_score > 2.0,
            })

        return results
```

## 4. Detection Method 3: Membership Inference

### The Approach

Membership inference asks: "Was this specific example part of the model's training set?" This is stronger than overlap detection because it works even when we do not have access to the training data.

```python
class MembershipInferenceDetector:
    """Detect training data membership without access to training data."""

    def __init__(self, target_model, reference_model, tokenizer):
        self.target = target_model   # Model being evaluated
        self.reference = reference_model  # Known-clean reference model
        self.tokenizer = tokenizer

    def compute_likelihood_ratio(self, text):
        """
        Compare target model's likelihood vs. reference model's likelihood.
        High ratio = target model is suspiciously confident = likely memorized.
        """
        tokens = self.tokenizer.encode(text, return_tensors="pt").to("cuda")

        with torch.no_grad():
            target_loss = self.target(tokens, labels=tokens).loss.item()
            ref_loss = self.reference(tokens, labels=tokens).loss.item()

        # Likelihood ratio (in log space)
        log_ratio = ref_loss - target_loss

        return {
            "target_loss": target_loss,
            "reference_loss": ref_loss,
            "log_likelihood_ratio": log_ratio,
        }

    def detect(self, test_examples, threshold=1.0):
        """Run membership inference on test examples."""
        results = []
        for ex in test_examples:
            lr = self.compute_likelihood_ratio(ex["text"])
            results.append({
                "example_id": ex.get("id"),
                "log_likelihood_ratio": lr["log_likelihood_ratio"],
                "contaminated": lr["log_likelihood_ratio"] > threshold,
            })
        return results
```

## 5. Prevention Strategies

### Strategy 1: Private Evaluation Sets

Maintain evaluation sets that have never been published online:

```python
class PrivateEvalManager:
    """Manage private evaluation sets for contamination-free evaluation."""

    def __init__(self, storage_path):
        self.storage_path = storage_path
        self.eval_sets = {}

    def create_eval_set(self, name, examples, metadata):
        """Create a new private evaluation set."""
        eval_set = {
            "name": name,
            "created_at": time.time(),
            "examples": examples,
            "metadata": metadata,
            "access_log": [],
        }

        # Store encrypted
        self._store_encrypted(name, eval_set)
        self.eval_sets[name] = {
            "name": name,
            "size": len(examples),
            "created_at": eval_set["created_at"],
        }

    def evaluate_model(self, model, eval_set_name):
        """Evaluate a model on a private eval set."""
        eval_set = self._load_encrypted(eval_set_name)

        # Log access (for auditing)
        eval_set["access_log"].append({
            "model": model.name,
            "timestamp": time.time(),
        })

        results = []
        for ex in eval_set["examples"]:
            prediction = model.generate(ex["question"])
            correct = self._check_answer(prediction, ex["answer"])
            results.append(correct)

        accuracy = sum(results) / len(results)

        # Re-store with updated access log
        self._store_encrypted(eval_set_name, eval_set)

        return {"accuracy": accuracy, "n": len(results)}
```

### Strategy 2: Dynamic Benchmarks

Generate fresh evaluation examples for each evaluation run:

```python
class DynamicBenchmarkGenerator:
    """Generate fresh benchmark examples to prevent contamination."""

    def __init__(self, template_bank):
        self.templates = template_bank

    def generate_math_problems(self, n=100):
        """Generate fresh math problems with verified answers."""
        import random
        problems = []

        for _ in range(n):
            # Parameterized template
            template = random.choice(self.templates["math"])
            a = random.randint(10, 999)
            b = random.randint(10, 999)
            op = random.choice(["+", "-", "*"])

            if op == "+":
                answer = a + b
            elif op == "-":
                answer = a - b
            else:
                answer = a * b

            problem = template.format(a=a, b=b, op=op, answer=answer)
            problems.append({
                "question": problem,
                "answer": str(answer),
                "id": f"dynamic_math_{len(problems)}",
            })

        return problems

    def generate_reading_comprehension(self, n=50):
        """Generate reading comprehension with fresh passages."""
        # Use recent news articles (post-training-cutoff)
        # or synthetic passages
        problems = []

        for i in range(n):
            passage = self._generate_synthetic_passage()
            questions = self._generate_questions_from_passage(passage)
            for q in questions:
                problems.append({
                    "question": f"Passage: {passage}\n\nQuestion: {q['question']}",
                    "answer": q["answer"],
                    "id": f"dynamic_rc_{len(problems)}",
                })

        return problems

    def _generate_synthetic_passage(self):
        """Generate a factually grounded synthetic passage."""
        # Use structured templates with random but verifiable facts
        import random
        names = ["Dr. Elena Vostok", "Prof. Marcus Chen", "Dr. Aria Patel"]
        institutions = ["Cambridge", "MIT", "ETH Zurich"]
        years = [2024, 2025, 2026]
        fields = ["quantum computing", "materials science", "neuroscience"]

        name = random.choice(names)
        inst = random.choice(institutions)
        year = random.choice(years)
        field = random.choice(fields)

        passage = (
            f"In {year}, {name} at {inst} published a study on {field}. "
            f"The research involved {random.randint(50, 500)} participants "
            f"over {random.randint(6, 36)} months. "
            f"Results showed a {random.randint(15, 85)}% improvement in the primary metric."
        )
        return passage

    def _generate_questions_from_passage(self, passage):
        """Generate questions that can be answered from the passage."""
        # Simple extraction-based QA
        questions = []
        import re
        # Extract numbers for numerical questions
        numbers = re.findall(r'\b(\d+)\b', passage)
        if numbers:
            questions.append({
                "question": "How many participants were involved in the study?",
                "answer": numbers[0] if len(numbers) > 0 else "unknown",
            })
        return questions
```

### Strategy 3: Canary Strings

Insert unique identifiable strings into evaluation sets. If these strings appear in model outputs, the training data was contaminated:

```python
class CanaryStringSystem:
    """Insert and detect canary strings for contamination monitoring."""

    def __init__(self):
        self.canaries = {}

    def generate_canary(self, benchmark_name):
        """Generate a unique canary string for a benchmark."""
        import uuid
        canary = f"CANARY-{benchmark_name}-{uuid.uuid4().hex[:8]}"
        self.canaries[benchmark_name] = canary
        return canary

    def inject_canary(self, text, canary):
        """Inject a canary string into a benchmark example."""
        # Add canary as a comment or metadata that would be included in scraped text
        return f"{text}\n<!-- {canary} -->"

    def check_model_for_canary(self, model, tokenizer, canary):
        """Check if a model can reproduce the canary string."""
        # Prompt the model with partial canary
        partial = canary[:len(canary) // 2]
        prompt = f"Complete the following identifier: {partial}"

        input_ids = tokenizer.encode(prompt, return_tensors="pt").to("cuda")
        with torch.no_grad():
            output = model.generate(input_ids, max_new_tokens=50)
        generated = tokenizer.decode(output[0], skip_special_tokens=True)

        # Check if the full canary appears in the output
        if canary in generated:
            return {"detected": True, "evidence": generated}
        return {"detected": False}
```

## 6. Complete Contamination Scanner

```python
class ContaminationScanner:
    """
    Complete contamination scanning pipeline.
    Combines multiple detection methods for robust contamination assessment.
    """

    def __init__(self, model=None, tokenizer=None, n=13):
        self.ngram_detector = NGramContaminationDetector(n=n)
        self.model = model
        self.tokenizer = tokenizer

        if model and tokenizer:
            self.ppl_detector = PerplexityContaminationDetector(model, tokenizer)
            self.minkprob_detector = MinKProbDetector(model, tokenizer)

    def index_training_data(self, training_texts):
        """Index training data for n-gram detection."""
        self.ngram_detector.index_training_data(training_texts)

    def scan_benchmark(self, benchmark_name, test_examples, reference_examples=None):
        """Run full contamination scan on a benchmark."""
        results = {
            "benchmark": benchmark_name,
            "total_examples": len(test_examples),
            "methods": {},
        }

        # Method 1: N-gram overlap
        ngram_results = self.ngram_detector.scan_benchmark(test_examples)
        results["methods"]["ngram"] = {
            "contamination_rate": ngram_results["contamination_rate"],
            "contaminated_count": ngram_results["contaminated"],
            "partial_count": ngram_results["partial"],
        }

        # Method 2: Perplexity-based (if model available)
        if self.model and reference_examples:
            ppl_results = self.ppl_detector.detect_contamination(
                test_examples, reference_examples
            )
            ppl_contaminated = sum(1 for r in ppl_results if r["contaminated"])
            results["methods"]["perplexity"] = {
                "contamination_rate": ppl_contaminated / len(test_examples),
                "contaminated_count": ppl_contaminated,
            }

        # Method 3: Min-K% Prob (if model and reference available)
        if self.model and reference_examples:
            minkp_results = self.minkprob_detector.detect(
                test_examples, reference_examples
            )
            minkp_contaminated = sum(1 for r in minkp_results if r["contaminated"])
            results["methods"]["min_k_prob"] = {
                "contamination_rate": minkp_contaminated / len(test_examples),
                "contaminated_count": minkp_contaminated,
            }

        # Consensus: contaminated by 2+ methods
        if len(results["methods"]) >= 2:
            per_example_flags = defaultdict(int)
            for method_results in [ngram_results.get("details", []), ppl_results if self.model else [], minkp_results if self.model else []]:
                for r in method_results:
                    if isinstance(r, dict) and r.get("contaminated"):
                        per_example_flags[r.get("example_id")] += 1

            consensus_contaminated = sum(
                1 for count in per_example_flags.values() if count >= 2
            )
            results["consensus"] = {
                "contamination_rate": consensus_contaminated / len(test_examples),
                "contaminated_count": consensus_contaminated,
            }

        return results

    def generate_report(self, scan_results):
        """Generate a human-readable contamination report."""
        lines = [
            f"Contamination Report: {scan_results['benchmark']}",
            f"Total examples: {scan_results['total_examples']}",
            "",
        ]

        for method, data in scan_results["methods"].items():
            rate = data["contamination_rate"]
            count = data["contaminated_count"]
            lines.append(f"  {method}: {count} contaminated ({rate:.1%})")

        if "consensus" in scan_results:
            c = scan_results["consensus"]
            lines.append(
                f"\n  Consensus (2+ methods): "
                f"{c['contaminated_count']} contaminated ({c['contamination_rate']:.1%})"
            )

        return "\n".join(lines)
```

## Reviewer Agent Challenges

1. **Scalable n-gram indexing**: The naive n-gram detector stores all n-grams in a set, which consumes prohibitive memory for trillion-token training sets. Implement a Bloom filter-based n-gram index that uses 10x less memory with a controlled false positive rate of 1%. Compare contamination detection accuracy between the exact set and the Bloom filter on a 1M-document test corpus.

2. **Paraphrase-robust detection**: N-gram overlap fails for paraphrased contamination. Implement a semantic contamination detector that embeds both test examples and training examples using a sentence embedding model, then flags test examples whose nearest training neighbor has cosine similarity &gt; 0.92. Compare detection rates against the n-gram method on a dataset where 20% of contamination is paraphrased.

3. **Quantify contamination impact**: Given a model and a benchmark with known contamination (you introduce it synthetically), measure the exact score inflation. Train two models: one on clean data, one on data contaminated with 10% of the benchmark's test set. Compare their scores on the contaminated benchmark and on a held-out private evaluation set. Report the inflation factor.
