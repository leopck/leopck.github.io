Text-only LLM serving routes requests based on two factors: KV cache locality and queue depth. Multimodal requests break this model. A video request requires encoding 30 frames through a ViT, producing 17,000+ visual tokens before the LLM can begin. An audio request needs Whisper-style preprocessing that is CPU-bound. These preprocessing stages have different hardware affinities and latencies than the LLM decode phase.

Dynamo extends its routing framework to handle multimodal workloads by introducing modality-aware routing, dedicated encoder pools, and scheduling algorithms that overlap encoding with LLM execution. This post covers the full architecture.

---

## 1. The Multimodal Routing Problem

### Why Text Routing Fails for Multimodal

Standard Dynamo routing optimizes for:

$$W^* = \arg\min_i \left[ T_{\text{prefill}}(\text{uncached}(P, W_i)) + T_{\text{queue}}(W_i) \right]$$

For multimodal requests, the TTFT equation becomes:

$$\text{TTFT}_{\text{mm}} = T_{\text{encode}}(\text{media}) + T_{\text{transfer}}(\text{visual\_tokens}) + T_{\text{prefill}}(\text{uncached}) + T_{\text{queue}}$$

The encoding term $T_{\text{encode}}$ dominates for video and audio. A 10-second video at 3 fps = 30 frames, each requiring ~5ms of ViT encoding = 150ms total. But if the encoding GPU is saturated, the wait time can be seconds.

### The Three-Stage Pipeline

Multimodal serving decomposes into three stages, each with different hardware requirements:

```
Stage 1: Media Encoding      (GPU: compute-bound, ViT/Whisper)
    |
    v  [visual_tokens tensor, GPU-to-GPU transfer]
    |
Stage 2: LLM Prefill         (GPU: compute-bound, matrix multiplications)
    |
    v  [KV cache populated]
    |
Stage 3: LLM Decode          (GPU: memory-bound, KV cache reads)
```

Each stage can run on different GPUs. Dynamo manages this heterogeneous pipeline.

## 2. Encoder Pool Architecture

### Dedicated Encoder GPUs

Dynamo allocates a pool of GPUs specifically for media encoding. These GPUs run vision encoders (ViT) and audio encoders (Whisper) but not the LLM:

```python
class EncoderPoolManager:
    """Manage a pool of GPUs dedicated to media encoding."""

    def __init__(self, encoder_gpu_ids, encoder_configs):
        self.encoders = {}

        for gpu_id in encoder_gpu_ids:
            self.encoders[gpu_id] = EncoderWorker(
                gpu_id=gpu_id,
                models={
                    "vision": load_vit(encoder_configs["vision"], device=f"cuda:{gpu_id}"),
                    "audio": load_whisper(encoder_configs["audio"], device=f"cuda:{gpu_id}"),
                },
                max_batch_encode=16,  # Max images per encoding batch
            )

        self.queue = PriorityQueue()  # Priority by deadline (SLO)
        self.gpu_loads = {gpu_id: 0 for gpu_id in encoder_gpu_ids}

    def submit_encoding(self, request):
        """Submit a media encoding request to the least-loaded encoder GPU."""
        # Select encoder GPU
        encoder_gpu = min(self.gpu_loads, key=self.gpu_loads.get)

        # Estimate encoding time
        encode_cost = self._estimate_cost(request)
        self.gpu_loads[encoder_gpu] += encode_cost

        # Submit to encoder worker
        future = self.encoders[encoder_gpu].encode_async(request)

        return EncodingHandle(
            future=future,
            encoder_gpu=encoder_gpu,
            estimated_time=encode_cost,
        )

    def _estimate_cost(self, request):
        """Estimate encoding cost in milliseconds."""
        cost = 0
        for media_item in request.media:
            if media_item.type == "image":
                cost += 5  # ms per image
            elif media_item.type == "video":
                cost += media_item.num_frames * 1.6  # ms per frame (batched)
            elif media_item.type == "audio":
                cost += media_item.duration_seconds * 2.8  # ms per second of audio
        return cost
```

### Encoder Worker

Each encoder worker handles batching and execution for a single GPU:

```python
class EncoderWorker:
    """Encoder worker running on a dedicated GPU."""

    def __init__(self, gpu_id, models, max_batch_encode):
        self.gpu_id = gpu_id
        self.models = models
        self.max_batch = max_batch_encode
        self.pending_queue = []
        self.lock = threading.Lock()

    def encode_async(self, request):
        """Submit for async encoding. Returns a Future."""
        future = concurrent.futures.Future()
        with self.lock:
            self.pending_queue.append((request, future))
        return future

    def run_batch_loop(self):
        """Continuous loop that batches and processes encoding requests."""
        while True:
            # Collect pending requests
            with self.lock:
                if not self.pending_queue:
                    time.sleep(0.001)  # 1ms polling
                    continue
                batch = self.pending_queue[:self.max_batch]
                self.pending_queue = self.pending_queue[self.max_batch:]

            # Group by modality for efficient batching
            image_batch = []
            video_batch = []
            audio_batch = []

            for request, future in batch:
                for media in request.media:
                    if media.type == "image":
                        image_batch.append((media, request, future))
                    elif media.type == "video":
                        video_batch.append((media, request, future))
                    elif media.type == "audio":
                        audio_batch.append((media, request, future))

            # Process each modality
            if image_batch:
                self._encode_images(image_batch)
            if video_batch:
                self._encode_videos(video_batch)
            if audio_batch:
                self._encode_audio(audio_batch)

    def _encode_images(self, batch):
        """Batch-encode images through ViT."""
        pixel_values = torch.stack([
            preprocess_image(item.data) for item, _, _ in batch
        ]).to(f"cuda:{self.gpu_id}", dtype=torch.float16)

        with torch.no_grad():
            features = self.models["vision"](pixel_values)

        for i, (media, request, future) in enumerate(batch):
            media.encoded_features = features[i].detach()

    def _encode_videos(self, batch):
        """Encode video frames through ViT."""
        for media, request, future in batch:
            frames = extract_frames(media.data, fps=media.target_fps)
            # Batch all frames through ViT
            pixel_values = torch.stack([
                preprocess_image(f) for f in frames
            ]).to(f"cuda:{self.gpu_id}", dtype=torch.float16)

            with torch.no_grad():
                # Process in sub-batches if too many frames
                all_features = []
                for i in range(0, len(pixel_values), self.max_batch):
                    chunk = pixel_values[i:i + self.max_batch]
                    features = self.models["vision"](chunk)
                    all_features.append(features)
                features = torch.cat(all_features, dim=0)

            media.encoded_features = features.detach()
```

## 3. Modality-Aware Routing

### The Routing Decision

When a multimodal request arrives, the router must decide:
1. Which encoder GPU processes the media
2. Which LLM GPU runs prefill and decode
3. Whether to overlap encoding with prefill of the text portion

```python
class MultimodalRouter:
    """Route multimodal requests considering encoder and LLM placement."""

    def __init__(self, encoder_pool, llm_workers, kv_cache_index):
        self.encoder_pool = encoder_pool
        self.llm_workers = llm_workers
        self.kv_cache_index = kv_cache_index

    def route(self, request):
        """Route a multimodal request."""
        has_media = len(request.media) > 0

        if not has_media:
            # Text-only: standard KV-aware routing
            return self._route_text_only(request)

        # Step 1: Estimate encoding cost
        encode_cost = self.encoder_pool._estimate_cost(request)

        # Step 2: Check for KV cache overlap on LLM workers
        # (system prompt may be cached even for multimodal requests)
        text_hash = hash_prompt_prefix(request.text_tokens)
        cache_overlaps = self.kv_cache_index.get_overlaps(text_hash)

        # Step 3: Score each LLM worker
        best_worker = None
        best_score = float("inf")

        for worker in self.llm_workers:
            # Effective TTFT = encode_time + transfer_time + prefill_time + queue_time
            overlap = cache_overlaps.get(worker.id, 0)
            uncached_tokens = request.total_tokens - overlap

            encode_time = encode_cost  # Same regardless of LLM worker
            transfer_time = self._estimate_transfer(request, worker)
            prefill_time = uncached_tokens / worker.prefill_throughput
            queue_time = worker.current_queue_depth * worker.avg_step_time

            total_ttft = encode_time + transfer_time + prefill_time + queue_time

            if total_ttft < best_score:
                best_score = total_ttft
                best_worker = worker

        # Step 4: Select encoder GPU closest to chosen LLM worker
        encoder_handle = self._select_encoder(request, best_worker)

        return RoutingDecision(
            encoder_gpu=encoder_handle.encoder_gpu,
            llm_worker=best_worker,
            estimated_ttft=best_score,
        )

    def _estimate_transfer(self, request, worker):
        """Estimate visual token transfer time from encoder to LLM GPU."""
        num_visual_tokens = sum(
            estimate_visual_tokens(m) for m in request.media
        )
        token_bytes = num_visual_tokens * 4096 * 2  # d_model * dtype_bytes
        # NVLink: ~600 GB/s; PCIe: ~32 GB/s
        bandwidth = self._get_bandwidth(request.encoder_gpu, worker.gpu_id)
        return token_bytes / bandwidth

    def _select_encoder(self, request, target_llm_worker):
        """Select encoder GPU with best connectivity to target LLM worker."""
        best_encoder = None
        best_bw = 0

        for encoder_gpu in self.encoder_pool.encoders:
            bw = self._get_bandwidth(encoder_gpu, target_llm_worker.gpu_id)
            load = self.encoder_pool.gpu_loads[encoder_gpu]

            # Prefer high bandwidth and low load
            score = bw / (1 + load / 100)
            if score > best_bw:
                best_bw = score
                best_encoder = encoder_gpu

        return best_encoder
```

### Encoding-Prefill Overlap

A key optimization: start prefill of the text portion while the encoder is still processing the media. The text tokens before the first `<image>` placeholder can be prefilled immediately:

```python
class OverlappedPipeline:
    """Overlap media encoding with text prefill."""

    def execute_request(self, request, encoder_gpu, llm_worker):
        # Find the position of first <image> token
        first_image_pos = find_first_media_token(request.text_tokens)

        # Phase 1: Start encoding (async) and prefill text prefix (parallel)
        encode_future = self.encoder_pool.submit_encoding(request)

        if first_image_pos > 0:
            # Prefill text tokens before the first image
            text_prefix = request.text_tokens[:first_image_pos]
            llm_worker.prefill_partial(text_prefix)

        # Phase 2: Wait for encoding to complete
        visual_tokens = encode_future.result()

        # Phase 3: Inject visual tokens and continue prefill
        remaining_tokens = request.text_tokens[first_image_pos:]
        llm_worker.prefill_with_visual(
            visual_tokens, remaining_tokens
        )

        # Phase 4: Begin decode
        return llm_worker.start_decode(request)
```

The overlap saves:

$$T_{\text{saved}} = \min(T_{\text{encode}}, T_{\text{prefill\_prefix}})$$

For a request with a 500-token system prompt before the first image, and 150ms of encoding time, the overlap saves $\min(150\text{ms}, 50\text{ms}) = 50\text{ms}$.

## 4. Video-Specific Handling

### Frame Extraction Strategies

Not all video frames need to be encoded. The frame extraction strategy balances visual fidelity with computational cost:

```python
class VideoFrameExtractor:
    """Extract frames from video with various strategies."""

    def __init__(self, strategy="uniform", max_frames=32):
        self.strategy = strategy
        self.max_frames = max_frames

    def extract(self, video_bytes, target_fps=None):
        """Extract frames from video."""
        import cv2
        import numpy as np

        # Decode video
        video = cv2.VideoCapture(io.BytesIO(video_bytes))
        native_fps = video.get(cv2.CAP_PROP_FPS)
        total_frames = int(video.get(cv2.CAP_PROP_FRAME_COUNT))
        duration = total_frames / native_fps

        if self.strategy == "uniform":
            # Evenly spaced frames
            frame_indices = self._uniform_sample(total_frames)
        elif self.strategy == "keyframe":
            # Extract only keyframes (scene changes)
            frame_indices = self._keyframe_sample(video, total_frames)
        elif self.strategy == "fps":
            # Fixed FPS sampling
            sample_fps = target_fps or 2
            step = max(1, int(native_fps / sample_fps))
            frame_indices = list(range(0, total_frames, step))
        elif self.strategy == "adaptive":
            # More frames for high-motion segments
            frame_indices = self._adaptive_sample(video, total_frames)

        # Limit to max_frames
        if len(frame_indices) > self.max_frames:
            step = len(frame_indices) / self.max_frames
            frame_indices = [frame_indices[int(i * step)] for i in range(self.max_frames)]

        # Extract selected frames
        frames = []
        for idx in frame_indices:
            video.set(cv2.CAP_PROP_POS_FRAMES, idx)
            ret, frame = video.read()
            if ret:
                frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
                frames.append(Image.fromarray(frame))

        video.release()
        return frames, frame_indices

    def _uniform_sample(self, total_frames):
        if total_frames <= self.max_frames:
            return list(range(total_frames))
        step = total_frames / self.max_frames
        return [int(i * step) for i in range(self.max_frames)]

    def _keyframe_sample(self, video, total_frames):
        """Extract frames at scene changes using frame differencing."""
        keyframes = [0]  # Always include first frame
        prev_frame = None

        for idx in range(0, total_frames, max(1, total_frames // 200)):
            video.set(cv2.CAP_PROP_POS_FRAMES, idx)
            ret, frame = video.read()
            if not ret:
                continue

            frame_gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
            frame_small = cv2.resize(frame_gray, (64, 64))

            if prev_frame is not None:
                diff = np.mean(np.abs(frame_small.astype(float) - prev_frame.astype(float)))
                if diff > 30:  # Threshold for scene change
                    keyframes.append(idx)

            prev_frame = frame_small

        return keyframes
```

### Video Token Budget

Video generates far more visual tokens than images. A cost-aware token budgeting system:

```python
class VideoTokenBudget:
    """Manage visual token budget for video requests."""

    def __init__(self, max_visual_tokens=8192, model_max_len=32768):
        self.max_visual_tokens = max_visual_tokens
        self.model_max_len = model_max_len

    def compute_budget(self, request):
        """Compute the visual token budget for a video request."""
        text_tokens = len(request.text_tokens)
        output_budget = request.max_output_tokens

        # Available visual token budget
        available = self.model_max_len - text_tokens - output_budget
        budget = min(available, self.max_visual_tokens)

        if budget <= 0:
            raise ValueError("Not enough context window for visual tokens")

        return budget

    def select_frame_count(self, budget, patches_per_frame=576):
        """Determine how many frames fit in the budget."""
        max_frames = budget // patches_per_frame
        return max(1, max_frames)
```

## 5. Audio Preprocessing

### Whisper-Style Feature Extraction

Audio requests go through mel-spectrogram extraction before encoding:

```python
class AudioPreprocessor:
    """Extract features from audio for multimodal models."""

    def __init__(self, config):
        self.sample_rate = config.audio_sample_rate  # 16000
        self.n_mels = config.n_mels                  # 80 or 128
        self.chunk_length = config.chunk_length       # 30 seconds
        self.n_fft = config.n_fft                    # 400
        self.hop_length = config.hop_length          # 160

    def preprocess(self, audio_bytes):
        """Convert raw audio to mel spectrogram features."""
        import torchaudio

        # Decode audio
        waveform, sr = torchaudio.load(io.BytesIO(audio_bytes))

        # Resample if necessary
        if sr != self.sample_rate:
            resampler = torchaudio.transforms.Resample(sr, self.sample_rate)
            waveform = resampler(waveform)

        # Convert to mono
        if waveform.shape[0] > 1:
            waveform = waveform.mean(dim=0, keepdim=True)

        # Compute mel spectrogram
        mel_transform = torchaudio.transforms.MelSpectrogram(
            sample_rate=self.sample_rate,
            n_fft=self.n_fft,
            hop_length=self.hop_length,
            n_mels=self.n_mels,
        )
        mel = mel_transform(waveform)

        # Log-mel
        mel = torch.clamp(mel, min=1e-10).log10()
        mel = (mel + 4.0) / 4.0  # Normalize

        # Pad or truncate to expected length
        max_frames = self.chunk_length * self.sample_rate // self.hop_length
        if mel.shape[-1] < max_frames:
            mel = torch.nn.functional.pad(mel, (0, max_frames - mel.shape[-1]))
        else:
            mel = mel[:, :, :max_frames]

        return mel  # [1, n_mels, T]

    def compute_audio_tokens(self, duration_seconds):
        """Estimate number of audio tokens."""
        # Audio encoder typically downsamples by 2x temporally
        frames = duration_seconds * self.sample_rate // self.hop_length
        return frames // 2  # After encoder downsampling
```

## 6. Scheduling Mixed-Modality Batches

### Heterogeneous Batch Formation

When the system has a mix of text-only, image, video, and audio requests, the scheduler must form efficient batches:

```python
class MultimodalBatchScheduler:
    """Schedule batches of mixed-modality requests."""

    def __init__(self, max_batch_tokens, max_batch_size):
        self.max_tokens = max_batch_tokens
        self.max_batch = max_batch_size

    def form_batch(self, waiting_queue):
        """Form a batch from the waiting queue."""
        batch = []
        total_tokens = 0

        # Sort by priority: requests with completed encoding first
        sorted_queue = sorted(
            waiting_queue,
            key=lambda r: (
                0 if r.encoding_complete else 1,  # Encoded first
                r.arrival_time,                     # Then FIFO
            )
        )

        for request in sorted_queue:
            if len(batch) >= self.max_batch:
                break

            # Skip if encoding is not complete
            if not request.encoding_complete and request.has_media:
                continue

            # Compute effective token count
            req_tokens = self._effective_tokens(request)

            if total_tokens + req_tokens > self.max_tokens:
                continue  # Skip this request (too large)

            batch.append(request)
            total_tokens += req_tokens

        return batch

    def _effective_tokens(self, request):
        """Compute effective token count for scheduling."""
        text_tokens = len(request.text_tokens)
        visual_tokens = request.num_visual_tokens if request.has_media else 0
        audio_tokens = request.num_audio_tokens if request.has_audio else 0
        return text_tokens + visual_tokens + audio_tokens
```

### Priority-Based Encoder Scheduling

Encoder GPUs should prioritize requests whose LLM workers are idle (to minimize TTFT):

```python
class PriorityEncoderScheduler:
    """Prioritize encoding based on downstream LLM availability."""

    def __init__(self, encoder_pool, llm_workers):
        self.encoder_pool = encoder_pool
        self.llm_workers = llm_workers

    def compute_priority(self, request):
        """Higher priority = should be encoded sooner."""
        # Priority factors:
        # 1. Target LLM worker queue depth (lower = higher priority)
        target_worker = request.assigned_llm_worker
        queue_depth = target_worker.current_queue_depth if target_worker else 10

        # 2. Encoding cost (cheaper = higher priority, to clear queue faster)
        encode_cost = self.encoder_pool._estimate_cost(request)

        # 3. SLO deadline pressure
        time_remaining = request.slo_deadline - time.time()
        deadline_pressure = 1.0 / max(time_remaining, 0.1)

        # Combined score (lower = higher priority)
        priority = queue_depth * 0.3 + encode_cost * 0.3 - deadline_pressure * 0.4
        return priority
```

## 8. Complete Multimodal Routing System

```python
class DynamoMultimodalRouter:
    """
    Complete multimodal routing system for Dynamo.
    Coordinates encoders, LLM workers, and KV cache.
    """

    def __init__(self, config):
        # Encoder pool
        self.encoder_pool = EncoderPoolManager(
            encoder_gpu_ids=config.encoder_gpus,
            encoder_configs=config.encoder_configs,
        )

        # LLM workers
        self.llm_workers = [
            LLMWorker(gpu_ids=gpus, model=config.model_name)
            for gpus in config.llm_gpu_groups
        ]

        # KV cache index
        self.kv_index = KVCacheIndex()

        # Metrics
        self.routing_latencies = []

    def handle_request(self, request):
        """Route and execute a multimodal request."""
        start = time.perf_counter()

        # Classify request
        modality = self._classify_modality(request)

        if modality == "text":
            worker = self._route_text(request)
            return worker.execute(request)

        # Multimodal path
        # Step 1: Select encoder and LLM worker jointly
        routing = self._joint_route(request, modality)

        # Step 2: Start encoding
        encode_handle = self.encoder_pool.submit_encoding(request)

        # Step 3: Overlap with text prefill if possible
        if routing.can_overlap:
            routing.llm_worker.prefill_text_prefix(
                request.text_tokens[:routing.overlap_boundary]
            )

        # Step 4: Wait for encoding
        visual_tokens = encode_handle.result()

        # Step 5: Complete prefill with visual tokens
        routing.llm_worker.complete_prefill(request, visual_tokens)

        # Step 6: Begin decode
        result = routing.llm_worker.decode_to_completion(request)

        self.routing_latencies.append(time.perf_counter() - start)
        return result

    def _classify_modality(self, request):
        has_image = any(m.type == "image" for m in request.media)
        has_video = any(m.type == "video" for m in request.media)
        has_audio = any(m.type == "audio" for m in request.media)

        if has_video:
            return "video"
        elif has_audio:
            return "audio"
        elif has_image:
            return "image"
        return "text"

    def _joint_route(self, request, modality):
        """Jointly select encoder GPU and LLM worker."""
        best_combo = None
        best_ttft = float("inf")

        for encoder_gpu in self.encoder_pool.encoders:
            encode_time = (
                self.encoder_pool._estimate_cost(request) +
                self.encoder_pool.gpu_loads[encoder_gpu]
            )

            for worker in self.llm_workers:
                transfer = self._transfer_time(encoder_gpu, worker)
                prefill = request.total_tokens / worker.prefill_throughput
                queue = worker.queue_time_estimate()

                ttft = encode_time + transfer + prefill + queue

                if ttft < best_ttft:
                    best_ttft = ttft
                    overlap_boundary = self._find_overlap_boundary(request)
                    best_combo = RoutingDecision(
                        encoder_gpu=encoder_gpu,
                        llm_worker=worker,
                        estimated_ttft=ttft,
                        can_overlap=overlap_boundary > 0,
                        overlap_boundary=overlap_boundary,
                    )

        return best_combo

    def _find_overlap_boundary(self, request):
        """Find how many text tokens can be prefilled before media."""
        for i, token in enumerate(request.text_tokens):
            if token in (IMAGE_TOKEN, VIDEO_TOKEN, AUDIO_TOKEN):
                return i
        return len(request.text_tokens)

    def _transfer_time(self, encoder_gpu, worker):
        """Estimate tensor transfer time between GPUs."""
        # Same node (NVLink): ~600 GB/s
        # Cross node (InfiniBand): ~50 GB/s
        same_node = encoder_gpu // 8 == worker.primary_gpu // 8
        bandwidth = 600e9 if same_node else 50e9
        bytes_to_transfer = 18000 * 4096 * 2  # Worst case video
        return bytes_to_transfer / bandwidth
```
