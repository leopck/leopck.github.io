## Complete Gating Function

The gating network produces a probability distribution over experts, selects the top-k, and returns routing weights:

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class TopKGating(nn.Module):
    """Top-k expert gating with straight-through estimator."""

    def __init__(self, d_model, num_experts, top_k=2):
        super().__init__()
        self.num_experts = num_experts
        self.top_k = top_k
        self.gate = nn.Linear(d_model, num_experts, bias=False)

    def forward(self, x):
        """
        x: [batch * seq_len, d_model]
        Returns:
            gates: [batch * seq_len, top_k] — routing weights (sum to 1)
            indices: [batch * seq_len, top_k] — selected expert indices
            load: [num_experts] — fraction of tokens routed to each expert
        """
        # Router logits
        logits = self.gate(x)  # [N, num_experts]

        # Top-k selection
        topk_logits, topk_indices = logits.topk(self.top_k, dim=-1)

        # Normalize gates (softmax over selected experts only)
        topk_gates = F.softmax(topk_logits, dim=-1)  # [N, top_k]

        # Compute load for auxiliary loss
        # load[i] = fraction of tokens where expert i is in top-k
        # Use softmax probabilities (differentiable) not hard assignments
        router_probs = F.softmax(logits, dim=-1)  # [N, num_experts]
        load = router_probs.mean(dim=0)  # [num_experts]

        return topk_gates, topk_indices, load
```

## Complete Auxiliary Load-Balancing Loss

$$L_{\text{balance}} = \alpha \cdot N_{\text{experts}} \cdot \sum_{i=1}^{N_{\text{experts}}} f_i \cdot P_i$$

where $f_i$ = fraction of tokens dispatched to expert $i$ (hard assignment), $P_i$ = average router probability for expert $i$ (soft, differentiable).

```python
def load_balancing_loss(topk_indices, router_probs, num_experts, top_k):
    """
    Compute auxiliary load-balancing loss.

    topk_indices: [N, top_k] — hard expert assignments
    router_probs: [N, num_experts] — soft router probabilities
    Returns: scalar loss
    """
    N = topk_indices.shape[0]

    # f_i: fraction of tokens dispatched to expert i (non-differentiable)
    # Count how many tokens are assigned to each expert
    expert_mask = torch.zeros(N, num_experts, device=topk_indices.device)
    expert_mask.scatter_(1, topk_indices, 1.0)  # [N, num_experts]
    f = expert_mask.float().mean(dim=0)  # [num_experts]

    # P_i: average router probability for expert i (differentiable)
    P = router_probs.mean(dim=0)  # [num_experts]

    # Loss: N_experts * sum(f_i * P_i)
    # When perfectly balanced: f_i = top_k/N_experts for all i
    # The product f_i * P_i is minimized when load is uniform
    loss = num_experts * (f * P).sum()

    return loss
```

<Callout type="info" title="Why f * P Works">
$f_i$ is the hard fraction (non-differentiable — just counts). $P_i$ is the soft probability (differentiable — backpropagates to the router). The gradient flows through $P_i$ only. When expert $i$ is overloaded ($f_i$ high), the gradient pushes $P_i$ down, discouraging future tokens from routing there. The product $f \cdot P$ is minimized when both $f$ and $P$ are uniform across experts.
</Callout>

## Complete Capacity Factor Enforcement

Each expert can process at most $C \times \lceil N \times k / E \rceil$ tokens, where $C$ is the capacity factor:

```python
def enforce_capacity(topk_indices, topk_gates, num_experts, capacity_factor=1.25):
    """
    Enforce capacity limits per expert. Drop tokens that exceed capacity.

    Returns:
        dispatch_mask: [N, top_k] — 1 if token is dispatched, 0 if dropped
        dropped_count: number of tokens dropped
    """
    N, top_k = topk_indices.shape
    # Maximum tokens per expert
    tokens_per_expert = int(capacity_factor * N * top_k / num_experts)

    # Count tokens per expert
    expert_counts = torch.zeros(num_experts, dtype=torch.long,
                                 device=topk_indices.device)
    dispatch_mask = torch.ones_like(topk_gates)

    # Process tokens in order — first-come-first-served
    flat_indices = topk_indices.reshape(-1)  # [N * top_k]
    flat_mask = dispatch_mask.reshape(-1)

    for i in range(flat_indices.shape[0]):
        expert = flat_indices[i].item()
        if expert_counts[expert] >= tokens_per_expert:
            flat_mask[i] = 0.0  # Drop this token-expert assignment
        else:
            expert_counts[expert] += 1

    dispatch_mask = flat_mask.reshape(N, top_k)
    dropped = (dispatch_mask == 0).sum().item()

    return dispatch_mask, dropped
```

## The Complete MoE Layer (Copy-Paste Ready)

```python
class CompleteMoELayer(nn.Module):
    """Complete Mixture of Experts layer with all components."""

    def __init__(
        self,
        d_model=4096,
        d_ff=11008,
        num_experts=8,
        top_k=2,
        capacity_factor=1.25,
        balance_coeff=0.01,
    ):
        super().__init__()
        self.num_experts = num_experts
        self.top_k = top_k
        self.capacity_factor = capacity_factor
        self.balance_coeff = balance_coeff

        # Gating network
        self.gating = TopKGating(d_model, num_experts, top_k)

        # Expert FFNs (SwiGLU)
        self.experts = nn.ModuleList([
            SwiGLUExpert(d_model, d_ff) for _ in range(num_experts)
        ])

    def forward(self, x):
        """
        x: [batch, seq_len, d_model]
        Returns: (output, aux_loss)
        """
        B, S, D = x.shape
        x_flat = x.view(-1, D)  # [N, D] where N = B * S

        # 1. Gating: select top-k experts per token
        gates, indices, load = self.gating(x_flat)  # gates: [N, k], indices: [N, k]

        # 2. Load-balancing loss
        router_probs = F.softmax(self.gating.gate(x_flat), dim=-1)
        aux_loss = self.balance_coeff * load_balancing_loss(
            indices, router_probs, self.num_experts, self.top_k
        )

        # 3. Capacity enforcement
        dispatch_mask, dropped = enforce_capacity(
            indices, gates, self.num_experts, self.capacity_factor
        )
        gates = gates * dispatch_mask  # Zero out dropped assignments

        # 4. Expert computation (scatter-gather pattern)
        output = torch.zeros_like(x_flat)  # [N, D]

        for expert_idx in range(self.num_experts):
            # Find tokens assigned to this expert (across both top-k slots)
            expert_mask = (indices == expert_idx)  # [N, k] boolean
            if not expert_mask.any():
                continue

            # Gather tokens for this expert
            token_indices = expert_mask.any(dim=-1).nonzero(as_tuple=True)[0]
            expert_input = x_flat[token_indices]  # [num_tokens, D]

            # Run expert FFN
            expert_output = self.experts[expert_idx](expert_input)  # [num_tokens, D]

            # Weighted scatter back
            for k in range(self.top_k):
                k_mask = expert_mask[:, k]  # [N] boolean for this k-slot
                k_indices = k_mask.nonzero(as_tuple=True)[0]
                k_gates = gates[k_indices, k].unsqueeze(-1)  # [num_tokens_k, 1]
                # Find matching tokens in expert_output
                # (tokens in k_indices are a subset of token_indices)
                for idx, token_idx in enumerate(k_indices):
                    local_idx = (token_indices == token_idx).nonzero(as_tuple=True)[0][0]
                    output[token_idx] += k_gates[idx] * expert_output[local_idx]

        return output.view(B, S, D), aux_loss


class SwiGLUExpert(nn.Module):
    """Single SwiGLU FFN expert."""
    def __init__(self, d_model, d_ff):
        super().__init__()
        self.w1 = nn.Linear(d_model, d_ff, bias=False)
        self.w2 = nn.Linear(d_ff, d_model, bias=False)
        self.w3 = nn.Linear(d_model, d_ff, bias=False)

    def forward(self, x):
        return self.w2(F.silu(self.w1(x)) * self.w3(x))
```

## DeepSeek V3 Loss-Free Balancing (Alternative)

DeepSeek's approach eliminates the auxiliary loss entirely:

```python
class LossFreeBiasBalancing:
    """DeepSeek V3 style: bias terms updated outside gradient descent."""

    def __init__(self, num_experts, target_load=None, update_rate=0.01):
        self.biases = torch.zeros(num_experts)
        self.target = target_load or (1.0 / num_experts)
        self.rate = update_rate

    def adjust(self, expert_loads):
        """Call after each batch. expert_loads: [num_experts] fractions."""
        for i in range(len(self.biases)):
            if expert_loads[i] > self.target * 1.1:
                self.biases[i] -= self.rate  # Discourage overloaded expert
            elif expert_loads[i] < self.target * 0.9:
                self.biases[i] += self.rate  # Encourage underloaded expert

    def apply(self, router_logits):
        """Add biases to router logits before softmax."""
        return router_logits + self.biases.to(router_logits.device)
```

**Challenge 1**: Compute the auxiliary loss for 8 experts with top-2 routing where expert 0 gets 40% of tokens and expert 7 gets 5%.

Expected: $f_0 = 0.40$, $P_0 \approx 0.40$ (correlated). $f_7 = 0.05$, $P_7 \approx 0.05$. Loss $= 8 \times (0.40 \times 0.40 + \ldots + 0.05 \times 0.05)$. If perfectly balanced: $f_i = 0.25$, $P_i = 0.125$, loss $= 8 \times 8 \times 0.25 \times 0.125 = 2.0$. The imbalanced case produces loss above 2.0, penalizing the imbalance.
